STANDALONE PLAYER INTERACTION SCRIPT

Author: Kira
version: 1.0.0

This is a flexible customisable standalone script compatible with QBCore framework.

Dependencies: qb-target
	     dpemotes/rpemotes/customemotes

Installation: 
Drag and drop kira-interaction folder into your standalone folder or inside resource folder.
Ensure kira-interaction in the server.cfg
start the script and you're good to go.


You can customise the player's interaction distance interaction emotes in config.lua.
